# Topics Index

Curated learning paths that group related concepts into recommended study orders.

## Primary Learning Paths

These three paths are the main suggested routes through the current concept library.

- [Web Scalability: From Single Server to Production](../topics/web-scalability.md) — 11 concepts · ~6–8h · Learn the chronological growth path from one server to a redundant web architecture.
- [Distributed Systems Foundations](../topics/distributed-systems-foundations.md) — 7 concepts · ~5–6h · Build the reasoning model for CAP, consistency, async messaging, microservices, and distributed data design.
- [Production LLM Engineering](../topics/production-llm-engineering.md) — 12 concepts · ~7–9h · Study observability, prompt lifecycle control, evaluation, optimization, and durable LLM infrastructure.

## Focused Companion Paths

These narrower paths reuse concepts from the primary paths for a specific practical goal.

- [LLM Program Optimization with DSPy](../topics/llm-program-optimization-dspy.md) — 7 concepts · ~3–5h · Focus on DSPy signatures, modules, metrics, bootstrapping, ReAct, and production observability.
- [Reliability Engineering for AI Backends](../topics/reliability-engineering-for-ai-backends.md) — 9 concepts · ~5–7h · Apply scalability and reliability patterns to AI event pipelines, queues, storage, and data stores.
